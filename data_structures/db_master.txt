JSON Master Record
==================

{
    "ranking": [
        [1, "djicon", "djname", 1500000],
        [2, "djicon", "djname", 1500000],
        [3, "djicon", "djname", 1500000],
        ...
    ]
}


Details
-------
This data structure is for the master rankings, which is the overall rank and
score for each DJ.  It has a single key called "ranking" and is similar to the
ranking key of a disc record except it is a list of lists instead of a list of
dictionaries.


Special Values
--------------
None.


Notes
-----
This structure is generated automatically and should not be edited manually.
