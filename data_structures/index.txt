JSON Index
==========

{
    "name": {
        "timestamp": 0,
        "page": 0
    },
    ...
}

Details
-------
The index is a dictionary of all available disc, disc set, or mission names.
Each game mode has its own index, and the names within the index represents a
database in DJRivals.  The timestamp marks when the database was last updated.
Page indicates where the disc, disc set, or mission shows up in the ranking
section of the DJMAX site.

Special Values
--------------
None.

Notes
-----
This structure is generated automatically and should not be edited manually.
