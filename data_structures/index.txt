JSON Index
==========

{
    "name": {
        "page": 0,
        "timestamp": 0
    }
}

Details
-------
The index is a dictionary of all available names.  Each game mode has its own
index.  Each name key accesses a dictionary that contains the page where the
disc shows up in the ranking section of the DJMAX site.  The timestamp is a
value indicating when this particular database was updated.

Special Values
--------------
None.

Notes
-----
This structure is generated automatically and should not be edited manually.
