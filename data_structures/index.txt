JSON Index
==========

{
    "mode": {
        "name": {
            "timestamp": 0,
            "page": 0
        },
        ...
    },
    ...
}

Details
-------
The index is a dictionary of names of all discs, disc sets, and missions,
separated by game mode.  Each "name" is a database in DJRivals.  The "timestamp"
marks when a database was last updated, while "page" indicates where it appears
in the ranking section of the DJMAX site.

Special Values
--------------
None.

Notes
-----
This structure is generated automatically and should not be edited manually.
