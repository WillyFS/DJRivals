JSON Index
==========

{
    "mode": {
        "name": {
            "timestamp": 0,
            "page": 0
        },
        ...
    },
    ...
}


Details
-------
A dictionary of the names of all discs, disc sets, and missions, separated by
game mode.  Each "name" is a record in a database of DJRivals.  The "timestamp"
marks when a database was last updated, while "page" indicates where it appears
in the ranking section of the DJMAX site.


Special Values
--------------
None.


Notes
-----
This structure is generated automatically and should not be edited manually.
